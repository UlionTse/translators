*1.Description:*
-------------

Free & Easy translators for Python, such as Google, NetEase(Youdao), etc.

*2.Usage:*
----------

$ pip install translators

>>> import translators

>>> translators.google('Hello,World!')

'你好，世界！'

>>> translators.youdao('再见，世界！','zh-CN','ko')

'안녕, 세계야!'
